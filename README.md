# Password Strength Analyzer

This is my second Python project.

The program asks the user for a password and checks how strong it is based on a few common security rules.

## What It Checks

* Password length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Based on those checks, the program rates the password as:

* Weak
* Fair
* Strong
* Very Strong


## Why I Made This

I'm learning Python and cybersecurity, and I wanted to build something that uses loops, conditionals, string methods, and basic security concepts.
